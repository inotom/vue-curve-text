# vue-curve-text

A Vue.js component that makes text align with the SVG curve.

## options

| name       | content         | defaults |
|:-----------|:----------------|:---------|
| `text`     | display text    |          |
| `:width`   | svg box width   | `200`    |
| `:height`  | svg box height  | `50`     |
| `:r`       | curve height    | `25`     |
| `textid`   | text anchor id  |          |
| `:debug`   | show curve line | `false`  |
